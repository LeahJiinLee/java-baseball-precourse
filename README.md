# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 기능 구현 목록
###[기능 구현]
1. 컴퓨터 수 생성
2. 사용자 수 생성
3. 입력 받기
4. 스트라이크 판단
5. 볼 판단
6. 낫싱 판단
7. 결과 출력
8. 게임 시작 (1), 종료(2) 입력 처리 
 -> 종료시 게임 끝 출력

###[예외 처리]
1. 겹치는 수가 있을 때
2. 숫자 3개 미만으로 입력 시
3. 숫자가 아닌 문자 입력시
4. 1~9사이의 수가 아닐 시
5. 게임 시작, 종료 입력 1,2 이외의 수 입력시
